## 🚀 Comment démarrer ?

### Préréquis
1.  **Python 3.x** installé.
2.  Installer les dépendances générales :
    ```bash
    pip install -r requirements.txt
    ```

### Lancement d'un projet
Chaque projet est indépendant. Pour en lancer un, déplacez-vous dans son dossier et exécutez le script principal :

```bash
cd projects/NOM_DU_PROJET
python script_principal.py
```
